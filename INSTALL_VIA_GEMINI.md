# 🤖 Gemini CLI 초고속 자동 설치 프롬프트 (Git Clone 버전)

이 내용을 Gemini CLI에 복사해서 붙여넣으세요. 가장 빠르고 정확한 설치 방법입니다.

---

### 📋 복사할 프롬프트

안녕 Gemini! 안호영(An Ho Yong) 개발자의 **Fast Obsidian MCP** 서버를 내 컴퓨터에 설치해줘.

1. **저장소 클론**: `https://github.com/sdcom1214/Obsidian-for-Gemini-Cli.git`를 `C:\Users\%USERNAME%\fast-obsidian-mcp` 폴더에 클론해줘.
2. **설정 등록**: 클론한 폴더의 `src/index.js`를 내 `.gemini/settings.json`에 `obsidian`이라는 이름의 MCP 서버로 등록해줘.
3. **환경 변수**: `OBSIDIAN_VAULT_PATH`는 `C:\Gemini Project`로 설정해줘.

위 작업을 한 번에 완료해줘!
